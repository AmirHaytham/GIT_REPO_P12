## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/AmirHaytham/git-a-head.git

# Enter the project directory
cd git-a-head

# Install dependencies
pip install -r tools/requirements.txt

# Start with Day 1
cd exercises/01-basics
```
